왜 이걸 만들었나
😓 Before  —  기존 문제
✗매일 아침 사업부별 엑셀 수동 취합에 30~60분 소요
✗달성률·신장률 계산식을 매번 직접 입력, 오류 발생
✗경영기획 보고용 시각화 자료를 별도 PPT로 제작
✗전년/전월 비교 시 파일 두 개를 열어 수동 확인
✅ After  —  이 시스템으로 해결
✓전년·목표·실적 입력 → 전체 KPI 자동 계산
✓소계·합계 자동 집계, 달성률 3단계 색상 경보
✓시각화 탭에서 바차트·히트맵 즉시 확인
✓체크박스 선택으로 기간 비교 1초 전환
